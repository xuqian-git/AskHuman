# 通用设置

简体中文 | [English](./settings.en.md)

运行 `AskHuman --settings`（或在弹窗右上角点齿轮）打开设置界面，分三个 Tab：**通用**、**Agents**、**通信渠道**。本文介绍「通用」相关偏好；Agents 集成见 README 的「集成到 Agent 中」，渠道接入见 README 的「配置沟通渠道」。

## 通用

- **主题**：跟随系统 / 浅色 / 深色。
- **窗口置顶**：让弹窗始终保持在最前。
- **出现动画**（仅 macOS）：None / Document / Alert。
- **窗口效果**（仅 macOS 26+）：玻璃 / 模糊。
- **语音输入**（仅 macOS）：识别语言与触发快捷键。
- **回复历史保留条数**：默认 200；设为 `0` 则停止新增并清理已有记录。当已有条数超过上限时，会出现「立即清理」按钮可立刻裁剪。

## 回复历史

每次在弹窗或任一渠道完成的「发送」与你主动「取消」都会记录到本地，方便回答新问题时参考。系统触发的取消（超时 / 断连 / daemon 停止）不记录。

- **打开**：`AskHuman --history`（默认仅当前项目；加 `--all` 查看全部项目），或在弹窗右上角点「历史」按钮。窗口内也可用顶部下拉切换项目。
- **项目识别**：从命令运行目录向上找首个 `.git` 仓库根；没有 `.git` 则用当前目录。
- **清空**：历史窗口右上角「清空」可清「当前项目」或「全部项目」。
